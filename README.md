# pokedex
Create a responsive frontend only poke dex . We need to fetch the data from an external API and manage the response and render the data received on a clean UI. The fetching of the pokemon details can be done from the Official Poke API (Link) where you can fetch the required data using multiple API endpoints availaible.

instructions
to search about any pokemon enter its name or number on search window and then click on search button.
to signin click on the profile logo on left side and submit your details.
to open other sites of pokemon website click on the images at uppermost part .
